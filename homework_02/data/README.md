# Data Directory

## Required Files

Place your multispectral Landsat 8 image file here:

- **Image.tif** - GeoTIFF format with 6 bands (B2, B3, B4, B5, B6, B7)
  - Band Order: B2 (index 0), B3 (index 1), B4 (index 2), B5 (index 3), B6 (index 4), B7 (index 5)
  - This file is typically >50MB and is excluded from git tracking

## Data Source

- **Satellite**: Landsat 8 OLI
- **Bands**: B2, B3, B4, B5, B6, B7 (6 multispectral bands)
- **Format**: GeoTIFF with 6 bands

## Notes

This file is excluded from git due to its large size (>50MB). Please download or place your image file here before running the scripts.
