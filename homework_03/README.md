# Homework 3: Land Surface Temperature (LST) Analysis

## Overview

This homework assignment focuses on calculating and visualizing Land Surface Temperature (LST) from Landsat 5 TM satellite imagery using Google Earth Engine.

## Objectives

1. Load and visualize Landsat 5 TM imagery
2. Create natural RGB and false color composites
3. Calculate LST from thermal Band 6
4. Generate temperature maps and statistical analysis
5. Perform city-level analysis (Rasht, Iran)

## Files

### Main Notebook
- `homework_03_lst_analysis.ipynb` - Complete analysis notebook with all visualizations

### Supporting Scripts
- `scripts/load_landsat5_tm_band6.py` - Scene selection and metadata extraction
- `scripts/calculate_lst.py` - Full LST calculation and visualization
- `scripts/calculate_lst_celsius.py` - Celsius temperature map with histogram
- `scripts/calculate_lst_rasht.py` - City-level analysis for Rasht

### Data
- `data/selected_scene.json` - Scene metadata (generated by load script)

### Outputs
- `output/` - All generated visualizations and maps

## Usage

### Option 1: Jupyter Notebook (Recommended)

jupyter notebook homework_03_lst_analysis.ipynb

## LST Calculation Method

1. **DN to Radiance**: `L = RADIANCE_MULT_BAND_6 × B6 + RADIANCE_ADD_BAND_6`
2. **Radiance to Kelvin**: `T_K = K2 / ln((K1 × ε / L) + 1)` where ε = 0.95
3. **Kelvin to Celsius**: `T_C = T_K - 273.15`
