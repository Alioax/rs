# SLC (Scan Line Corrector) Error Images Directory

## Required Files

Place your SLC error correction images here:

- **SLC error.tif** - Image showing SLC error (typically >50MB, excluded from git)
- **SLC self-fill.tif** - Self-fill corrected image (typically >50MB, excluded from git)
- **SLC two band gap fill.tif** - Two-band gap fill corrected image (typically >50MB, excluded from git)

## Notes

These files are excluded from git due to their large size (>50MB). Please place your SLC correction images here before running the comparison scripts.
