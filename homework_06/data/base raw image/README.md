# Base Raw Image Directory

## Required Files

Place your base raw image file here:

- **543 stacked.tif** - Stacked Landsat bands 5, 4, 3 (B543 composite)
  - This file is typically >50MB and is excluded from git tracking
  - Used as the reference/base image for image-to-image registration

## Notes

This file is excluded from git due to its large size (>50MB). Please place your base raw image file here before running the registration scripts.
